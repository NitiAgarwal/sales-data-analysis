Sales Data Analysis Project

Project Overview  
This project is a simple data analysis of sales data using Python.  
It explores product-wise sales, region-wise performance, and monthly sales trends to extract meaningful insights.  

The goal is to understand how sales vary across different categories and time periods.  

Dataset Description  
The dataset contains the following columns:  

- Date → Transaction date  
- Product → Product name (Laptop, Mobile, Tablet, etc.)  
- Region → Sales region (North, South, East, West)  
- Sales_Amount → Total sales value  
- Quantity → Number of units sold  

Tools and Libraries Used  
- Python  
- Pandas  
- Matplotlib  
- Jupyter Notebook  

Key Analysis Performed  

1. Data Cleaning  
- Checked missing values  
- Verified data types  
- Converted Date column to datetime format  

2. Descriptive Statistics  
- Mean, median, min, max values analyzed  
- Basic summary of dataset  

3. Product-wise Sales  
- Total sales calculated for each product  
- Identified best-performing product  

4. Region-wise Sales  
- Compared sales across North, South, East, and West regions  
- Found top-performing region  

5. Monthly Sales Trend  
- Extracted month from date  
- Analyzed sales trends over time  

Visualizations  
The project includes the following charts:  

- Bar chart: Product-wise sales  
- Bar chart: Region-wise sales  
- Line chart: Monthly sales trend  

All graphs are clearly labeled for better understanding.  

Key Insights  

- Laptop generated the highest sales among all products  
- North region showed strong performance  
- January recorded the highest monthly sales  
- Sales vary significantly across months and regions  

How to Run This Project  

1. Open Jupyter Notebook  
2. Load the dataset  
3. Run all cells step by step  
4. View outputs and visualizations
